Proxel documentation 1.1

Controls

Left click on node: Select the node

Left click on node + Shift: Add the node to selected nodes

Move with left click on node: Move the selected nodes

Move with left click on background: Draw a selection rectangle that selects all nodes inside

Move with left click on pin: Draw a link from the clicked pin

Move with wheel click: Move on the canvas

Scroll up / down: Zoom in / Zoom out

Ctrl + C: Copy selected nodes

Ctrl + V: Paste copied nodes near the mouse

F: Focus the searchbar

Return: If the searchbar is focused, add the first node result to the canvas

T: If you are drawing a bool, number or string type link from an input pin, connect it to a bool, number or string node.

Delete / Return: Delete focused nodes (except start)



Pin types

● Execution pin : Used to create a flow of execution. When an output execution pin is activated, the linked execution pin also activates and executes the associated node.

● Boolean pin : Gets or receives a boolean value, "True" or "False".

● String pin : Gets or receives a string value (text).

● Numeric pin : Gets or receives a numeric value. All types of numbers are included (positive, negative, floating).

● Array pin : Gets or receives an array of values. An array can contain values of different types, be careful handling them.

● Neutral pin : Gets or receives any type of value. Like for arrays, be careful handling them.

Start

This node is executed first when the script is run. It cannot be added or deleted by the user, it is added automatically when the project is created.

Inputs:

None

Outputs:

●Execution output

Create Variable

Stores a value in memory and assigns it a name.

Inputs:

●Execution input

●NameVariable name

●ValueVariable value

Outputs:

●Execution output

Get Variable

Inputs:

●Execution input

●NameVariable name

Outputs:

●Execution output

●ValueVariable value

Delete Variable

Delete a value stored with Create Variable.

Inputs:

●Execution input

●NameVariable name

Outputs:

●Execution output

Forget Pin

During the execution of the script, the calculated values are bound to the corresponding pins. This node allows you to delete this value to free up memory, but the pin will be unusable until the pin node is rerun.

Inputs:

●Execution input

●PinPin with the value to delete

Outputs:

●Execution output
